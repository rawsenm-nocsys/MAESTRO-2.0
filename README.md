# MAESTRO-LITE
This is the LITE version of MAESTRO developed primarily for uploading to the BATON REVB and use with the ADAGIO EDU versions. This is a compact webapp capable of running on an RPi Zero 2W

This is a test for the auto-update functionality!!!!!!!!!!!!!!!

To run the auto-updater, paste this into the terminal while ssh'd into the RPi
journalctl -u maestro-update.service -n 50 --no-pager
